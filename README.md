# Luhn-Algorithmus Validator
###  Beschreibung
Dieses Konsolenprogramm überprüft, ob eine 12-stellige Nummer anhand des Luhn-Algorithmus gültig ist.

### Funktionen
- Liest eine 12-stellige Zahl vom Benutzer ein.
- Überprüft die Eingabe auf korrektes Format.
- Zerlegt die Zahl in einzelne Ziffern.
- Verdoppelt jede zweite Ziffer.
- Subtrahiert 9 von Ergebnissen größer als 9.
- Berechnet die Prüfsumme.
- Gibt aus, ob die Nummer gültig oder ungültig ist.
